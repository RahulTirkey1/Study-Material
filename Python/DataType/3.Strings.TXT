Slicing
----------------
I-> mystring="Hello World"
I-> mystring
O-> "Hello World"
I-> mystring[2:]           // Here : after indicates, slicing from which index number to start and till end.
O-> "llo World"
I-> mystring[:3]           //Here : before indicates from index 0 to index mentioned as argument.Here index 3 won't get included only from 0 to 2 will be included.
O-> "Hel"
I-> mystring[1:3]          //Part of that substring will be selected whose index is mentioned.
O-> "el"     
I-> mystring[::]           //It is prints everything. It is same as calling mystring in this example.
O-> "Hello World"
I-> mystring[::2]          //Prints the 0th index and next skips one and prints next and so on. It depends upon the number how much to skip.              
O-> 'HloWrd'
I-> mystring[::3]
O-> 'HlWl'
I-> mystring[1:8:2]        //Starting index:Last index:Step jump . So here from index 1 to index 8 and jump two steps.
O-> 'el o'
I-> mystring[::-1]
O-> 'dlroW olleH'          //Reverses a string.
