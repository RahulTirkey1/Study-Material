Suppose, we want to fillup the forms from the class without filling up manually from HTML. This is a case scenario if we fetch the values from an API.

1)setValue-We msut fill up all the formControl orelse it will throw an error. That's the only limitation.

app.component.ts,
import { Component } from '@angular/core';
import { FormGroup, FormControl } from '@angular/forms';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'app';
  registrationForm = new FormGroup({                  //Model for storing values from template to classes.
    userName: new FormControl('Vishwas'),
    password: new FormControl(''),
    confirmPassword: new FormControl(''),
    address: new FormGroup({
      city: new FormControl(''),
      state: new FormControl(''),
      postalCode: new FormControl('')
     }) 
  });
}loadAPIData() {
    this.registrationForm.setValue({          //Here we fillup all the formControl
      userName: 'Bruce',
      password: 'test',
      confirmPassword: 'test',
      address: {
        city: 'City',
        state: 'State',
        postalCode: '123456'
      }
    });
  }
}

2)patchValue- If we only want to fillup some of the formControls not all formControls then this approach is correct.

From app.component.ts,
 loadAPIData() {
   this.registrationForm.patchValue({
      userName: 'Bruce',
      password: 'test'
     });
 }
